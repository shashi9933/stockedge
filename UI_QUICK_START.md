**A:** Use the sidebar search field. Start typing the company name or symbol, and auto-complete suggestions will appear.

## Troubleshooting

### Charts not loading?
1. Check internet connection
2. Try refreshing the page (F5)
3. Clear browser cache
4. Try a different browser

### Symbol not found?
1. Ensure correct format (e.g., `RELIANCE.NS` for NSE)
2. Check if stock is active on the exchange
3. Try the symbol in Google Finance first

### Slow performance?
1. Close other browser tabs
2. Clear browser cache
3. Try a more recent browser version
4. Reduce date range for faster loading
